Armado y Desarmado
de equipos tecnológicos

El armado y desarmado de una PC se refiere al proceso de ensamblar y desmontar los componentes que conforman una computadora personal. Esto puede incluir la instalación y conexión de la placa base, la CPU, la memoria RAM, la tarjeta gráfica, el disco duro o SSD, la fuente de alimentación, entre otros componentes.

¿De qué
se compone el Armado?

Este se compone de varias seciones como: seccion de componentes, Preparacion del espacio, Ensamblaje, Conexiones y Pruebas

De que se compone
el Desarmado?

Este se compone de varias secciones como: Desconexion, Retiro de Componentes, Desmontaje de la placa base, Empaque seguro y Almacenamiento.

Etapa de Desarmado

1.Desconexión

Se apaga la PC y se desconecta de la fuente de alimentación.

2.Retiro de componentes

Se retiran los cables conectados a la placa base y a los demás componentes. Se retiran las tarjetas de expansión, la memoria RAM, la tarjeta gráfica y otros componentes.

3.Desmontaje de la Placa Base

Si es necesario, se retira la placa base del chasis.

4.Empaque Seguro y Almacenamiento

Se guardan los componentes en bolsas antiestáticas para prevenir daños por electricidad estática y Los componentes desmontados pueden ser almacenados para su uso futuro, reemplazo o actualización.

Etapa de Armado

1.Sección de Componentes

Se eligen los componentes necesarios para construir la PC, teniendo en cuenta factores como el rendimiento deseado, el presupuesto y la compatibilidad entre los elementos.

2.Preparación de Espacios

Se debe tener un espacio limpio y adecuado para trabajar, con herramientas y materiales necesarios a mano.

3.Ensamblaje

Se instala la placa base en el chasis de la carcasa, se monta la CPU en el zócalo correspondiente, se coloca la memoria RAM, se conectan los cables de alimentación y datos, se insertan las tarjetas de expansión como la tarjeta gráfica, y se conectan los dispositivos de almacenamiento y la fuente de alimentación.

4.Conexiones y Pruebas

Se desconectan los cables para la alimentacion electrica y la transmisión de datos para los componentes . Una vez ya listo se ensamblan los componentes y se enciende la PC
